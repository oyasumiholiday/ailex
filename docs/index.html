Ailex

A small typed language designed for AI to write. Contracts are executable examples; diagnostics disclose what's in scope; two verified backends.

Why an AI-first language

  • Contracts are executed. Every eg line runs on every check — on both backends. Break one and you get a structured report, not silence.
  • Diagnostics disclose scope. Type errors carry every usable name with its type as JSON. Measured: this turns an LLM's unknown-API guessing loop into a one-step fix (pass@k 80%→98–100%).
  • Syntax follows the model's prior. When lambda annotations were mandatory, every first-attempt failure was that parse error. The language was fixed, not the models: 50–63% → 100% first-attempt correctness.
  • One page teaches it. With only the primer in context, Claude Haiku and Claude Opus wrote 16/16 tasks correctly on the first attempt. Every claim above links to logged experiments in the repo.

Use it locally

git clone https://github.com/oyasumiholiday/ailex && cd ailex
node core/cli.ts run examples/gradebook.ax     # check + contracts + run (Node 23+)
npm test                                       # 89 conformance cases, both backends
